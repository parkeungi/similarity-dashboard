초기 설정 - 도메인으로 쉽게 접속하기

주의! 현재 localhost로 접속하셨습니다.
다른 PC에서 접속하려면 서버의 실제 IP 주소로 이 페이지에 접속해야 합니다.
예: http://192.168.x.x:4000/setup
0서버 IP 확인

서버 IP 주소 (자동 감지됨, 필요시 수정):

callsign.monitor:4000
1설정 파일 다운로드

아래 버튼을 클릭하여 설정 파일을 다운로드하세요.

2관리자 권한으로 실행

다운로드한 파일을 마우스 오른쪽 클릭 → "관리자 권한으로 실행"을 선택하세요.

3도메인으로 접속

설정 완료 후 아래 주소로 접속할 수 있습니다:

http://callsign.monitor:4000 - 관제사 화면
http://callsign.monitor:4000/admin - 관리자 화면
참고: 설정은 PC당 한 번만 하면 됩니다. 이미 설정한 PC에서는 바로 도메인으로 접속 가능합니다.
관제사 화면으로 이동 관리자 화면으로 이동
